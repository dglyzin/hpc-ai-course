# Занятие 1. Инфраструктура

## Адреса

- Юпитерхаб: https://jupyterhub.yarsu.ru (пользовательские контейнеры на узле `cnode07`)
- Управляющий узел для входа по ssh: `cnode07`
- Вычислительный узел (входа через ssh нет): `cnode08`
- Изнутри университетской сети узел `cnode07` имеет адрес `10.7.71.204`, узел `cnode08` имеет адрес `10.7.71.203`
- Короткие имена видны только изнутри кластера или из Юпитерхаба.

## Linux

- `ssh <имя пользователя>@<имя машины>` ssh-доступ
- `passwd` смена пароля. Пароль можно поменять только на управляющем узле `cnode07`
- `nano <имя файла>` текстовый редактор
- `cd <имя папки>` перейти в папку
- `ls -la`  посмотреть содержимое папки включая скрытые файлы (их имена начинаются с точки)

## SLURM

- `sinfo` список ресурсов
- `squeue` список задач в очереди
- `scancel <номер задачи>` остановить задачу
- `ctrl+C` два раза подряд - остановить интерактивную задачу
- `srun -w cnode07 hostname` запустить задачу с указанием узла
- `srun -w cnode08 --gpus-per-task=1 nvidia-smi -L`
- `srun -w cnode08 --gpus-per-task=2 sh /clusterhome/install/llamascripts/server_qwen_next_1.sh`
