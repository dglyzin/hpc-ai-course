\begin{frame}{ДОБАВЛЕНИЕ ВЫЗОВА ИНСТРУМЕНТОВ В ЦИКЛ}
    
    Мы уже делали цикл общения с LLM на Питоне:
    
    \begin{itemize}
        \item [-] сообщение пользователя
        \item [-] сообщение LLM
        \item [-] повторить
    \end{itemize}
    
    Добавляем действие:
    
    \begin{itemize}
        \item [-] сообщение пользователя
        \item [-] сообщение LLM
        \item [-] если сообщение LLM содержит вызов инструмента, то проверяем его на допустимость (напр. не удаляет данные из базы) и выполняем, затем ответ отправляем LLM
        \item [-] иначе выводим сообщение от LLM пользователю
        \item [-] повторить        
    \end{itemize}
    
    
\end{frame}  



 \begin{frame}{ПРИМЕР: ИНСТРУМЕНТ РАБОТЫ С БАЗОЙ ДАННЫХ}
    
    Что от нас требуется:
    
    \begin{itemize}
        \item [-] информация для LLM в системном промпте о том, что она может обращаться к базе данных через вызов инструмента и какие данные она должна подготовить (в данном случае строку запроса)
        \item [-] схема базы в системном промпте (пока база не слишком большая) или объяснение, как добыть данные о таблицах и полях
        \item [-] сам инструмент - функция, которая валидирует полученную от LLM строку запроса и отправит ее как sql-запрос к базе, а затем вернет ответ от базы
    \end{itemize}
    
    
\end{frame}  

\begin{frame}{ТИПОВАЯ ВАЛИДАЦИЯ}
    
   
    \begin{itemize}
        \item [-] безопасность: инструмент должен сам определять, какие операции допустимы, чтобы не повредить данные или не выдать конфиденциальную информацию (не полагаться на то, что LLM будет в точности следовать системному промпту)
        \item [-] размер ответа от базы: инструмент не должен допускать непреднамеренные ответы большого размера 
    \end{itemize}
    
    
\end{frame}

def execute_sql_query(query: str) -> List[Dict[str, Any]]:
    """
    Execute a SQL query and return results as a list of dictionaries.
    
    Only SELECT queries are allowed for security.
    Unbounded queries (without LIMIT) will automatically have LIMIT 20 applied.
    
    Args:
        query (str): SQL query to execute (must be a SELECT query)    
    Returns:
        List[Dict[str, Any]]: Query results as list of row dictionaries    
    Raises:
        ValueError: If query is not a SELECT statement
        Exception: If query execution fails
    """
